## 🌟 Features / Características

### 🇬🇧 English
- **Ollama AI Integration**: Local LLM chat processing with emotion detection and polylingual support.
- **Glassmorphism UI**: Modern, translucent interface with soft corners (24px) and premium shadows.
- **Smooth Chaining**: Support for up to 5 sequential speech bubbles for long responses.
- **Internal Thoughts**: Real-time analytical process visualization in cyan.
- **Performance Modes**: Low, Medium, and High settings to optimize CPU/GPU usage.
- **VHS Retro Effect**: Authentic analog noise, scanlines, and subtle jitter.

### 🇪🇸 Español
- **Integración con Ollama**: Procesamiento de chat local con detección de emociones y soporte políglota.
- **Interfaz Glassmorphism**: Diseño moderno y translúcido con bordes redondeados (24px) y sombras premium.
- **Encadenamiento Fluido**: Soporte para hasta 5 bocadillos de texto para respuestas largas.
- **Pensamientos Internos**: Visualización en tiempo real del razonamiento de la IA en color cian.
- **Modos de Rendimiento**: Ajustes Low, Medium y High para optimizar el uso de CPU/GPU.
- **Efecto Retro VHS**: Ruido analógico, líneas de escaneo y jitter sutil.

## 🛠 Installation / Instalación

Choose your preferred method / *Elige tu método preferido*:

### 1. One-click Script (Recommended)
**Linux/Mac:**
```bash
bash install.sh
```
**Windows/Any:**
```bash
python install.py
```

### 2. NPM
```bash
npm install
```

### 3. Manual Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 🚀 Execution / Ejecución

Start the avatar using any of these commands:
- **Bash Alias:** `makima`
- **NPM:** `npm start`
- **Python:** `python run.py`

## ⚙️ Configuration / Configuración

Access the **Settings Panel** (⚙️ icon) to adjust:
- **Language**: English / Spanish.
- **Performance**: Low (Battery Saver) / Medium / High.
- **Model**: Select from your local Ollama models.
- **Thought Mode**: None, Short, Deep, or Creative.
- **Persona**: Define how Makima speaks and thinks via System Prompt.
